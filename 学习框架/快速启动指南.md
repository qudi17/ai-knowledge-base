## 🎯 5 分钟快速开始

### 步骤 1：选择学习主题（1 分钟）

想一个你现在最想学的主题，例如：
- RAG 技术
- Agent 架构
- 向量数据库
- LangChain

### 步骤 2：创建学习计划（2 分钟）

1. 打开 Obsidian
2. 进入 `📚 学习框架/templates/`
3. 复制 `01-学习计划模板.md`
4. 重命名为 `[主题名]-学习计划.md`
5. 填写学习目标

### 步骤 3：开始学习（2 分钟）

1. 打开学习材料（论文/文档）
2. 用 `02-学习笔记模板.md` 记笔记
3. 专注学习 25 分钟（一个番茄钟）

## 📋 完整流程示例

### 示例：学习 RAG 技术

#### 第 1 天：输入阶段（1-2 小时）

```
✅ 创建学习计划
   → 使用模板：01-学习计划模板.md
   → 填写：为什么学 RAG、预期产出、时间预算

✅ 主动阅读笔记
   → 使用模板：02-学习笔记模板.md
   → 记录：核心问题、关键概念、公式代码

✅ 绘制概念图
   → 工具：Obsidian Canvas / Excalidraw
   → 内容：RAG 架构、组件关系
```

#### 第 2 天：内化阶段（30 分钟）

```
✅ 费曼验证
   → 使用模板：03-费曼验证模板.md
   → 任务：向自己解释 RAG、回答关键问题

✅ 识别盲区
   → 标记不理解的地方
   → 补充学习
```

#### 第 3-7 天：应用阶段（2-3 小时）

```
✅ 选择应用场景
   → 使用模板：04-应用场景模板.md
   → 选择：构建知识库问答系统

✅ 启动实践项目
   → 创建项目计划
   → 开始编码/实验
```

#### 第 14 天：回顾阶段（30 分钟）

```
✅ 学习回顾
   → 使用模板：05-学习回顾模板.md
   → 反思：收获、困难、改进

✅ 知识沉淀
   → 整理最佳实践
   → 记录常见陷阱
```

## 💡 常见问题

### Q1: 每个主题都要这么完整吗？

**答**：不一定。根据重要性调整：

| 重要性 | 流程 | 时间投入 |
|--------|------|---------|
| ⭐⭐⭐ 核心 | 完整 4 阶段 | 10-20 小时 |
| ⭐⭐ 重要 | 输入 + 内化 + 应用 | 5-10 小时 |
| ⭐ 了解 | 输入 + 内化 | 1-2 小时 |

---

### Q2: 没时间怎么办？

**答**：微学习模式：

```
每天 15 分钟：
- 5 分钟：阅读
- 5 分钟：费曼验证
- 5 分钟：思考应用
```

**关键**：持续性 > 单次时长

---

### Q3: 如何坚持？

**答**：三个技巧：

1. **绑定习惯**：学习后立刻记笔记
2. **公开承诺**：告诉别人你要学什么
3. **追踪进展**：用仪表盘看进步

**Jarvis 可以**：
- 每天提醒学习
- 每周询问进展
- 每月生成报告

---

### Q4: 学完就忘怎么办？

**答**：间隔重复 + 实践应用

```
遗忘曲线：
1 天后：忘记 70%
→ 解决方案：1 天后复习
→ 解决方案：3 天后复习
→ 解决方案：7 天后复习
→ 解决方案：实践应用
```

**实践过的知识，遗忘率降低 80%**
