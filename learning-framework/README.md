> **学习不是输入，而是转化。知识只有被应用，才真正属于你。**

```
📥 输入 (Learn)   →  系统性吸收

   ↓
🎯 输出 (Apply)   →  现实世界应用
   ↓
📊 反馈 (Review)  →  持续改进
```

## 💡 关键技巧

### 1. 费曼技巧

**核心**：如果能简单解释，就是真懂了

**步骤**：
1. 选择概念
2. 假装教学（向 10 岁孩子解释）
3. 发现卡壳点
4. 返回学习
5. 简化语言

**Jarvis 可以**：引导提问，识别盲区

---

### 2. 间隔重复

**原理**：根据遗忘曲线安排复习

**时间表**：
```
学习后 → 1 天 → 3 天 → 7 天 → 14 天 → 30 天
```

**Jarvis 可以**：设置提醒，到期通知

---

### 3. 项目驱动

**原则**：学以致用，用中学

**流程**：
```
学习 → 选场景 → 做项目 → 得反馈 → 再学习
```

**Jarvis 可以**：追踪进展，记录问题

## 🎓 示例学习路径

### 示例：学习 RAG 技术

**第 1 天**（2 小时）：
```
✅ 创建学习计划（5 分钟）
✅ 主动阅读笔记（30 分钟）
✅ 绘制概念图（25 分钟）
✅ 开始实践项目规划（30 分钟）
✅ 费曼验证准备（30 分钟）
```

**第 2 天**（30 分钟）：
```
✅ 费曼验证（20 分钟）
✅ 识别盲区（5 分钟）
✅ 补充学习（25 分钟）
```

**第 3-7 天**（3 小时）：
```
✅ 实践项目（2 小时）
✅ 每周回顾（30 分钟）
✅ 知识沉淀（30 分钟）
```

**第 14 天**（30 分钟）：
```
✅ 学习回顾（30 分钟）
✅ 规划下一步（15 分钟）
```
